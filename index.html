Question 1

What is Bootstrap??

Bootstrap is a resampling method in statistics that involves creating multiple resamples of a dataset to estimate the variability of a statistic. It is a non-parametric technique that does not make assumptions about the underlying distribution of the data, making it applicable to a wide range of situations.

Question 2

Is it necessary to use Bootstrap in your web projects?

Using Bootstrap in web projects is not necessary, but can be helpful for saving time in creating responsive and mobile-friendly websites.

Question 3

Give examples of some websites that currently use or used Bootstrap in the past

These websites and many others have used Bootstrap in the past: Twitter, Spotify, Airbnb, Udemy, Vogue, NASA, The New York Times, Intel, Expedia and PayPal.
